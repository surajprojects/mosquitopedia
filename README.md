# Mosquitopedia

Mosquitopedia is a fun, community-driven web app inspired by a social media trend of documenting killed mosquitoes with names, time, and stories.

Think of it as an encyclopedia — but for mosquitoes.

Tagline - Find any mosquito, anytime.

## Project Status

Under Active Development

This project is currently in its early stages.  
Core features are being built step by step, and more functionality will be added over time.

## Idea & Vision

Mosquitopedia allows users to
- Search and explore mosquito entries
- Read stories behind each mosquito kill
- View recent and related mosquitoes
- Contribute their own mosquito entries after registration

The long-term goal is to grow this into a fun, interactive, and mobile-friendly experience.

## Planned Features

### Public
- Search mosquitoes
- View mosquito details (image, story, time, location)
- Browse recent & trending entries

### Authenticated Users
- Email/Password authentication
- Username setup
- Add mosquito entries with images
- Create or select mosquito families (global & unique)

### Engagement (Planned)
- Likes on mosquito entries
- Contribution stats
- Testimonials / fun community feedback

## Mosquito Entry Data

Each mosquito entry contains
- Name
- Location
- Time of death (date & time)
- Killer name (username)
- Mosquito family
- Image
- Story
- (Optional later) External reference link
